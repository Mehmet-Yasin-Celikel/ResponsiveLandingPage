# Responsive Landing Page 🎨

A fully responsive personal portfolio landing page built with pure HTML5 and CSS3 — no frameworks, no JavaScript.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

🔗 **Live Demo:** [View Site](https://mehmet-yasin-celikel.github.io/ResponsiveLandingPage)

---

## Features

- **Fully responsive** — separate CSS files for desktop, tablet (768px), and mobile (500px)
- **Multi-section layout** — Hero, Projects, Courses, and Contact sections
- **Contact form** — Name, email, and message fields
- **Font Awesome icons** — Social media links in the footer
- **Google Fonts** — Open Sans typography

---

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and semantics |
| CSS3 | Styling and layout |
| Media Queries | Responsive breakpoints (768px, 500px) |
| Font Awesome 5 | Icons |
| Google Fonts | Typography |
